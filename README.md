# mango #

My mango config uses `SUPER` as the main key, `rofi -show drun` as the application launcher(SUPER+space), `foot` as the terminal emulator (SUPER+Return), in the config, you can also see at the top that it uses `swaybg -i ~/Downloads/background.jpg`, expecting the background to be in the `/home/user/Downloads` folder. It also uses waybar and starts pipewire as a service (`pipewire &`)
